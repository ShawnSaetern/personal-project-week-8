## 3. Project Retrospective

I was able to work on Jira by organizing tasks that helped make the project easier to manage. 
Breaking the work into smaller tasks made it clearer what needed to be done and helped track progress during the sprint. 
Planning out features like the dashboard and budget alerts also helped me understand what the main parts of the project are going to be. 
A challenge, was figuring out how long each task might take since some features are more complicated than they look at first. 
If I were to do another sprint, I would probably try to break bigger tasks into even smaller ones so progress can be tracked better. 
Overall, this sprint helped me understand how backlog planning and sprint work together in an Agile project.
